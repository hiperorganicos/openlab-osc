hiperorganicos-cupula-dos-povos
===============================

1o passo: Baixar OscGroups

  Windows e Mac: http://code.google.com/p/oscgroups/downloads/list
  Ubuntu: http://devolts.org/oscgroupslinux.tar.gz

2o passo: Instalação

  Windows: descompactar arquivos
  Mac: descompactar e copiar binarios para pasta 
  Ubuntu: descompactar e copiar binarios para pasta /usr/bin
  $ sudo cp oscpack/bin/* /usr/bin/.
  $ sudo cp oscgroups/bin/* /usr/bin/.
